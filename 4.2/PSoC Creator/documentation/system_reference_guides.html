PSoC Creator System Reference Guide
Design-Wide Components
Design-wide components allow configuration of chip resources without using schematic instantiated PSoC Creator components. They are designed to give low-level access to chip resources that can be used independent of schematic component configurations.  
cy_boot
The cy_boot component provides the system functionality for a project to give better access to chip resources. The functions are not part of the component libraries but may be used by them. You can use the function calls to reliably perform needed chip functions.

    cy_lfclk
    The PSoC 4 Low-Frequency Clock (cy_lfclk) component provides the application interface to configure various low-frequency clocks available in PSoC 4. The functions are not part of any component libraries but may be used by them. The component also provides functions to configure watchdog timers present in the device.

    cy_dmac
    The PSoC 4 Direct Memory Access (DMA) Controller (cy_dmac) component transfers data to and from memory, components, and registers. These transfers occur independent of the CPU and can transfer up to 65,536 data elements.
